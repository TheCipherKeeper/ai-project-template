# Указатель

Методология рассчитана на одного человека, который формулирует одну задачу за
раз, и программных агентов, выполняющих остальную работу.

1. [`START.md`](START.md) — прочитать один раз: модель, роли и основные правила.
2. [`WORKFLOW.md`](WORKFLOW.md) — открыть для выполнения любой задачи.
3. [`ARCHITECTURE.md`](ARCHITECTURE.md) — открыть при изменении границ системы,
   контрактов или структуры кода.
4. [`OPERATIONS.md`](OPERATIONS.md) — открыть при деплое, миграции, откате или
   выпуске версии.
5. [`REFERENCE.md`](REFERENCE.md) — команды, статусы, триггеры и форматы.

Обычная продуктовая задача требует только `START.md` и `WORKFLOW.md`. Остальные
документы читаются по условию, а не загружаются агентом всегда.
