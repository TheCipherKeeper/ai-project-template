# Восстановление pipeline и глобального лока — справочник

Активная задача имеет `run_id`, owner, target commit, phase, `acquired_at` и
`heartbeat_at`. Git-маркер `[~]` остаётся источником истины; heartbeat не даёт
права другому процессу автоматически забрать лок.

Watchdog только сообщает о просрочке. Координатор сверяет PR, commit, CI и
deploy, затем идемпотентно продолжает тот же run либо переводит задачу в
`automation-failed`. Автоматический захват чужого просроченного lease запрещён.
Аварийное освобождение лока выполняется отдельным PR с evidence и причиной.
