# Повторы и остановка автономного цикла — справочник

Значения по умолчанию:

| Цикл | Максимум |
|---|---:|
| исправление реализации | 3 |
| исправление adversarial-теста | 2 |
| conformance-фикс | 2 |
| security-фикс | 1 |
| повтор инфраструктурного вызова | 3 |

Каждая попытка фиксирует finding ID/fingerprint, действие, commit и результат.
Цикл немедленно останавливается, если одна находка повторилась после заявленного
исправления, два агента требуют взаимоисключающих изменений либо дифф ходит
туда-обратно. Результат — `retry-exhausted` или `automation-failed`, evidence и
hold; молчаливое продолжение запрещено.
