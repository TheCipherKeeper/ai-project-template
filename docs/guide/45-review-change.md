# Этап 45 — Выполнить независимое ревью

**Цель:** проверить изменение свежими контекстами после rule-гейта.

Тестировщик, ревьюер и безопасник выполняются последовательно по `ROLES.md` и
полномочиям `POLICY.md`. Каждая находка получает стабильный ID, adversarial-
verify и один исход: false positive, фикс с re-gate или hold с задачей. Ошибка
обязательной роли блокирует pipeline по `RISK.md`.
