# Свидетельства выполнения

CI публикует неизменяемый `.evidence/TASK-NNNN.json` по схеме
`<methodology-repo>/schemas/evidence.schema.json`. Средство проверки связывает его с
машинной задачей, закреплённой версией методологии и переданным CI-коммитом.

Поле `source` указывает на журнал проверки, `reviewer` — на независимый
вызов проверяющего, `artifact` содержит хеш, а `retained_until` задаёт срок, до
которого CI обязан сохранять свидетельство и связанные журналы. Секреты и персональные
данные сюда не записываются; исполняемые артефакты хранятся в CI, а не в Git.
