# AGENTS.md — автономный компонент

Отдельная программа, не являющаяся сервисом системы. Рабочий цикл —
`<methodology-repo>/docs/WORKFLOW.md`, архитектурные границы —
`<methodology-repo>/docs/ARCHITECTURE.md`.

## Правила

- Выполняй только текущую задачу из `<hub>/BACKLOG.md`.
- Работай через `feat/TASK-NNNN-<slug>` и PR; прямой commit в `main` запрещён.
- Форма поставки (`container`, `cli` или другая) фиксируется в
  `docs/ARCHITECTURE.md`.
- Компонент не подключается к брокеру и не выдаётся за сервис.
- Код, тесты и документация изменяются вместе.
- `.env`, файлы блокировки зависимостей и зависимости не меняй без разрешения.
- Placeholder/TODO не считается реализацией.
- Все диаграммы оформляются только блоками `mermaid`; текстовые псевдосхемы и
  другие языки диаграмм запрещены.

## Проверка

Зафиксируй точные format/lint/typecheck, test и build команды выбранного стека в
этом файле. Перед слиянием выполни их, средство проверки методологии и применимые проверки
из `<methodology-repo>/docs/REFERENCE.md`.

Интеграция — слияние со свёрткой коммитов; поставляемый результат идентифицируется SHA коммита или
digest. Тег выпуска создаётся отдельной задачей.
