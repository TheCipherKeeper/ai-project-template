## Потребление

> Манифест вызываемых маршрутов клиентского API **сервиса-шлюза**. **Обязателен**:
> его обязательная проверка № 14 сверяет контрактными тестами с архитектурой
> **сервиса-шлюза**.
> Маршруты API должны существовать и совпадать по версии с тем, что документировано
> в сервисе-шлюзе. Интерфейс потребляет только его API.

| Сервис-шлюз | Маршрут | Метод | Версия | Формат ответа | Назначение |
|---|---|---|---|---|---|
| `<gateway>` | `/v1/<endpoint>` | GET / WS | v1 | JSON | … |
| `<gateway>` | `/v1/<endpoint>` | POST | v1 | JSON | … |
