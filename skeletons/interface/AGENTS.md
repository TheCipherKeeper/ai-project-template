# AGENTS.md — интерфейс

Клиентское приложение. Общие полномочия и запреты, рабочий цикл, архитектурный
канон и правила поставки — в закреплённой методологии: `<methodology-repo>/AGENTS.md`
и `<methodology-repo>/docs/`. Ниже только локальные правила интерфейса.

## Локальные правила

- Интерфейс обращается только к зарегистрированному клиентскому сервис-шлюзу.
- Не храни секреты в клиентской сборке и репозитории.
- Доступность, состояния загрузки, отсутствия данных и ошибки, а также проверка
  прав входят в независимую проверку применимых изменений.
- Обновляй UI, тесты и `docs/ARCHITECTURE.md` вместе.

## Проверка

```bash
pnpm lint && tsc --noEmit
pnpm test
pnpm build
```

Дополнительно запускаются средство проверки методологии и применимые проверки
из `<methodology-repo>/docs/REFERENCE.md`.
